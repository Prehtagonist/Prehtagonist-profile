# Portfolio Images

Place all portfolio images in this directory. The following images are referenced in the portfolio:

## Required Images

- `avatar.jpg` - Your profile picture for the hero section
- `campaign-placeholder.jpg` - Placeholder for campaign screenshots (used 3 times)
- `nft-placeholder.jpg` - Placeholder for NFT/project images (used 2 times)
- `leaderboard-placeholder.jpg` - Placeholder for leaderboard position image

## Image Specifications

- Use high-quality images (at least 400x400 pixels recommended)
- Optimize for web (compress without losing quality)
- Use JPG or PNG formats
- Keep file sizes under 200KB for optimal loading speed
